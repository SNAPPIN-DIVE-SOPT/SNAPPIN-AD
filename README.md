# Snappin’

무드 큐레이션을 통해 다양한 스냅 작가와 고객을 정교하게 매칭하여 나의 가장 소중한 순간을 가장 나답게 담아냅니다.

## 핵심 UX 키워드

> #### 연결하는 : 내가 원하는 무드를 가진 작가를 찾아 매칭
>
> #### 빛나는 순간 : 인생에 가장 빛나는 순간을 포착
>
> #### 다채로운 : 다양한 무드의 스냅 사진을 모두 스냅핑에서
